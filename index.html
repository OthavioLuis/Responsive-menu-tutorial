Código em HTML

Primeiro criei o corpo padrão do HTML, fiz duas listas iguais dentro do header, a primeira fica com o Display:Block, e a segunda com Display:None, pois a segunda só irá aparecer com o width de 550px, fiz as Classes e Id's com nomes fáceis para todos entenderem.

Código em CSS

Criei as estilizações para o Header ficar no centro, com limite até onde aumentar, ajustei cada lista para seu display, organizei os locais e o visual dos botões, e criei a responsividade para quando o width estiver com 550px, as listas irem para suas configurações.
E para o eeader sobressair como no exemplo deste site, basta colocar no header: position: fixed; width: 100%; e por ultimo dar um padding para não ficar colado com as laterais do site.
Criei uma classe dentro da responsividade de 550px chamada abrir, ela é importante para que o JavaScript, onde veremos a seguir.

Código em JavaScript

Criei uma função como podem ver no HTML, chamada menu(), ao clicar(onclick) no menu lateral(hambúrguer), a classe abrir mostra a segunda lista que ocultamos e troca a imagem para um X, e ao clicar novamente no X toda alteração é desfeita, fazendo com que a lista desaparecesse e voltasse a imagem do menu lateral(hambúrguer).
